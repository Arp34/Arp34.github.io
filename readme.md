# The Shattered Riffs - Official Website

A modern, responsive single-page website for the rock band **The Shattered Riffs**, featuring events, band information, and a multimedia gallery.

##Preview

The website includes:
- **Header** - Logo and navigation (desktop + mobile hamburger menu)
- **Hero Section** - Visual impact with call-to-action
- **Events** - Upcoming tour dates in a table format
- **About** - Band description and values
- **Multimedia** - Image gallery with grid layout
- **Footer** - Contact information

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. No build step required - pure HTML/CSS/JS

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, CSS Grid, Custom Properties
- **JavaScript** - Mobile navigation

## Responsive Design

The site is fully responsive with:
- Desktop navigation
- Mobile hamburger menu
- Adaptive grid layouts
